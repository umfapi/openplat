## 5.2、短信

**简要描述** 

- 短信接口

**请求URL** 

- `http://ip:port/support/sms`

**请求方式**

- POST 

**请求参数说明** 

| **名称**     | **参数** | **必选** |         | **说明**     |
| ------------ | -------- | ------------ | ------- | ------------ |
| 服务请求流水 | rpid     | 是           | max(16) | 服务调用标识 |
| 品牌编号     | brand_no | 是           | max(8) |              |
| 手机号       | phone    | 是           | max(11) |              |
| 短信内容     | message  | 是           | max(60) |              |

**请求参数示例**

```json
  {    
    "rpid": "20200000001020102030",
    "brand_no": "LV000001",
    "phone": "13991391399",
    "message": "测试"
  }
```

**返回参数说明** 

| **名称**     | **参数** | **必选** | **说明**     |
| ------------ | -------- | ------------ | ------------ |
| 服务请求流水 | rpid     | 是           | 服务调用标识 |
| 返回码       | ret_code | 是           | 0000-成功    |
| 返回信息     | ret_msg  | 是           |              |

**返回参数示例**

```json
  {    
    "rpid": "20200000001020102030",
    "ret_code": "0000",
    "ret_msg": "交易成功"
  }
```
